# blog-backernd

## Project setup

### Python setup
```
pip install -r requirements.txt
```

#### Compiles and hot-reloads for development
```
python manage.py runserver
```

#### Open browers 
```
127.0.0.1:8000
```

### Docker setup
```
docker-compose build
```
#### Compiles and hot-reloads for development
```
docker-compose up
```
#### Open browers  
```
127.0.0.1:8000
```
